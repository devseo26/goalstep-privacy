# GoalStep Privacy Policy

**Effective Date: April 2026**

## 1. Purpose of Collection and Use of Personal Information
GoalStep (hereinafter "the App") collects and uses minimal information to smoothly provide users with goal tracking, chore management, and a reward system.
- **Service Provision**: User account identification (login/signup), synchronization of goals and reward history (using PocketBase server).
- **Prevention of Fraudulent Use and App Stability**: May be utilized for error (bug) logging and security checks.

## 2. Items of Personal Information Collected
The App does not collect sensitive information that can identify individuals without authorization, other than the data entered by the user.
- **Required items**: Login email and password, username (nickname)
- **Optional items**: Unique device identifier (for auto-login purposes)

## 3. Retention and Use Period of Personal Information
If a user requests account deletion or the deletion of personal information, the corresponding user's personal information is destroyed without delay.

## 4. Provision and Entrustment to Third Parties
The App, in principle, does not provide users' personal information to external parties. However, it uses PocketBase (an open-source backend server) to store data and communicate for feature implementation, utilizing encrypted communication during this process.

## 5. User Rights and Methods of Exercise
Users can request to view, modify, or destroy (delete account) their own or their children's personal information at any time, and can directly delete data within the system.
